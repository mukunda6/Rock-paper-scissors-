# DP on Two Arrays / Strings

These problems use Dynamic Programming where the state depends on
two indices (i, j).

Most problems follow the pattern:

dp[i][j] → answer using first i elements of array1
           and first j elements of array2

Time Complexity usually:
O(n × m)

Space Complexity:
O(n × m) or optimized O(m)

---

## Problems Included

1. Longest Common Subsequence (LCS)
2. Longest Common Substring
3. Edit Distance
4. Distinct Subsequences
5. Wildcard Matching
6. Shortest Common Supersequence
7. Minimum Insertions to Make Palindrome
8. Minimum ASCII Delete Sum

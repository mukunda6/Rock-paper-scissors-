# Greedy Algorithms

This repository contains solutions to important **Greedy Algorithm problems** commonly asked in coding interviews.

Greedy algorithms make the **locally optimal choice at each step** with the hope of finding the **global optimum**.

---

# Greedy Pattern

A greedy algorithm works when:

- The problem has **optimal substructure**
- A **local optimum leads to global optimum**

Typical strategy:

1. Sort input
2. Choose best option
3. Move forward

# Complexity

Greedy problems usually involve sorting.

```
Time Complexity: O(n log n)
Space Complexity: O(1) or O(n)
```
